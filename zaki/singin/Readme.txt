How to install:
---------------
1. Extract 'phpmysimplelogin' folder in archive to your 'htdocs' folder, eg: 'C:\xampp\htdocs\phpmysimplelogin'
2. Open PHPMyAdmin, usualy 'http://localhost/phpmyadmin/' (for local computer)
3. Create a database, name it 'phpmysimplelogin' without quotes.
4. Click 'Import', browse 'database.sql' then click 'Go'.
5. Try to open the script, 'http://localhost/phpmysimplelogin/'
6. For demo, username is 'admin' and password is 'admin'.
